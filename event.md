    **Post new event**
----


* **URL**

  /api/events

* **Method:**

  `POST`
```
  **Required:**
 eventName=[string]
 eventType=[string]
 eventDate=[string]
 eventDescription=[string]
 eventBudget=[string]
 location=[string]
 agenda=[string]
 user_id=[integer]
  ```

  **Example Body**
  ```json
  
{
  "firstName": "test",
  "lastName": "test",
  "email": "test@email.com",
  "password": "password",
  "yearOfBirth": 1976,
  "jobTitle": "test",
  "company": "test",
  user_id: 1
}
  
  ```
